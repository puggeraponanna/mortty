# mortty

> This project is vibe coded ✨

Mortty is a skeleton for a next-generation terminal emulator built in Rust using `winit` and `wgpu`. 
It aims to decouple text rendering into GPU-accelerated pipelines for seamless and fluid terminal updates unlike traditional terminals.

## Prerequisites
- Rust toolchain (via rustup)

## Running the project

Simply build and run using Cargo:
```bash
cargo run
```
